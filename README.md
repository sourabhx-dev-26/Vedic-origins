# Vedic-origins

A small web project (testing/demo) built with HTML, CSS, TypeScript, and JavaScript.

## About

This repository contains frontend code for a web project. The current purpose is for testing and development — use this README as a starting point and expand the documentation as the project grows.

## Features

- Static HTML layout
- Styling with CSS
- Interactive behavior using JavaScript and TypeScript

## Tech stack

- HTML
- CSS
- TypeScript
- JavaScript

## Getting started

1. Clone the repository:

   git clone https://github.com/sourabhx-dev-26/Vedic-origins.git

2. Open the project folder in your preferred editor.

3. If the project uses a build step (TypeScript), install dependencies and build:

   npm install
   npm run build

   If there is no build setup, you can open the HTML files directly in your browser for testing.

## Development

- Edit source files (HTML/CSS/TS/JS) in the repo.
- If using TypeScript, compile changes before testing in the browser (or use a watcher such as `tsc --watch`).
